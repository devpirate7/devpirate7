MOTION

An object is in MOTION its distance from another object is changing i.e if it changes position relative to a reference point. A REFERENCE POINT is a place or object used for comparison to determine if something in motion.

SPEED

The distance traveled by an object per unit if time is called its SPEED. An object can travel with different speeds such as: 

  30km/h      90km/h       150km/h

ACCELERATION

The rate of change of speed is called Accelration. Increment in the speed of an object is called ACCELERATION . Decrement in the speed of an object is called DECELERATION .

The total distance traveled by an object divided by the total time taken to travel, gives the AVERAGE SPEED of the object.
